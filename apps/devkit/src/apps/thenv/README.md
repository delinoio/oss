# Thenv Mini App

This directory hosts the Devkit mini app with the stable id `thenv`.

## Route Contract
- `/apps/thenv`

## Runtime Status
- Active mini app integrated with the thenv backend through Devkit Connect RPC transport.
- Provides scope selection, bundle list/detail/activate/rotate flows, policy editor, and audit viewer.
- Secret file content is masked by default and revealed only on explicit user action.
