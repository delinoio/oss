# Projects Overview

This page provides a high-level public catalog of projects in the Delino OSS monorepo.

## Active Projects

- [`cargo-mono`](cargo-mono): Cargo subcommand tooling for Rust monorepo workflows.
- [`nodeup`](nodeup): Rust-based Node.js runtime manager.
- [`derun`](derun): Terminal-fidelity run execution and MCP bridge tool.
- [`with-watch`](with-watch): Command rerun watcher for inferred or explicit filesystem inputs.
- [`dexdex`](dexdex): Connect RPC-first desktop + server orchestration platform.
- `mpapp`: Expo React Native mobile app.
- `devkit`: Next.js micro-app host platform.
- `devkit-commit-tracker`: Scaffolded Devkit mini app route reserved for future commit metrics workflows.
- `devkit-remote-file-picker`: Scaffolded Devkit mini app route reserved for future signed URL upload workflows.
- `thenv`: Secure `.env` sharing system across CLI, server, and web console.
- `public-docs`: Mintlify public documentation app.
