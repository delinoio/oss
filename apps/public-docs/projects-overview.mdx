# Projects Overview

This page provides a high-level public catalog of projects in the Delino OSS monorepo.

## Active Projects

- `cargo-mono`: Cargo subcommand tooling for Rust monorepo workflows.
- `nodeup`: Rust-based Node.js runtime manager.
- `derun`: Terminal-fidelity run execution and MCP bridge tool.
- `devmon`: Automation daemon with menu bar controls.
- `mpapp`: Expo React Native mobile app.
- `devkit`: Next.js micro-app host platform.
- `devkit-commit-tracker`: Commit Tracker web/API/collector project.
- `devkit-remote-file-picker`: Signed URL based file picker mini app.
- `thenv`: Secure `.env` sharing system across CLI, server, and web console.
- `public-docs`: Mintlify public documentation app.

## Canonical Internal Contracts

Authoritative implementation contracts are maintained in repository documents under `docs/project-*.md`.
Use this public catalog for orientation, and use internal project docs for precise engineering and operational details.
