# Delino OSS Public Docs

Welcome to the public documentation surface for the Delino OSS monorepo.

This site provides a curated, user-facing layer of documentation that complements internal project contracts stored in `docs/`.

## Start Here

- Read [Getting Started](getting-started) for local setup and editing workflow.
- See [Projects Overview](projects-overview) for the current public project catalog.
- Review [Documentation Lifecycle](documentation-lifecycle) for update rules between `docs/` and `apps/public-docs`.
- Open [Nodeup](nodeup), [Cargo Mono](cargo-mono), [Derun](derun), and [With Watch](with-watch) from the top navigation for dedicated project guides.

## Scope

This Mintlify app focuses on externally consumable guidance and stable product-facing information.
Deep implementation contracts remain in repository project documents under `docs/project-*.md`.
